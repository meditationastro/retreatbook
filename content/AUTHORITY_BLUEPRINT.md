# Clue UX Authority Blueprint (Editor Notes)

## Publishing rules
- Every blog post must link to:
  1) its pillar hub (Blog, Answers, Guides, Retreats)
  2) 2 related articles

## File format
Blog posts live in: content/blog/*.md
Use frontmatter:
title, description, date, category, tags
